# PostgreSQL 笔记

---

## 简介

[PostgreSQL](https://www.postgresql.org/) 是一款功能强大的开源数据库。

### 版本

PostgreSQL 社区采用「5 年生命周期」策略。

PostgreSQL 版本号由主要版本和次要版本组成。

> [!info] 
> 
> PostgreSQL 10.0 之前的版本由 3 个数字组成，例如 `9.6.19`，其中 `9.6` 是主要版本，`19` 是次要版本。
