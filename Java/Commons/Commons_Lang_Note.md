# Commons Lang 笔记

---

[Commons Lang](https://commons.apache.org/proper/commons-lang/) 是对 [Java](../Java_Note.md) 语言增强类库。

## 版本

主流版本是 3.x，一般 `jar` 名称是`commons-lang3`。

Commons 依赖 [JDK](../Java_Note.md#java_jdk) 版本：

> [!info] 
> 
> 只列出最早更换 JDK 版本的 Commons 版本，未列出的 3.x 版本可依据此列表中的版本推断。
> 
> 详细版本列表请参考：[Apache Commons Lang Release Notes](https://commons.apache.org/proper/commons-lang/changes.html)

| 版本 |  发布日期  | 最低要求的 [JDK](../Java_Note.md#java_jdk) 版本 |
|:----:|:----------:|:-----------------------------------------------:|
| 2.6  | 2011-01-16 |                    Java 1.3                     | 
| 3.0  | 2011-07-18 |          [JDK5](../Java_Note.md#JDK5)           |
| 3.2  | 2014-01-01 |          [JDK6](../Java_Note.md#JDK6)           |
| 3.6  | 2017-06-08 |          [JDK7](../Java_Note.md#JDK7)           |
| 3.9  | 2019-04-09 |          [JDK8](../Java_Note.md#JDK8)           |
